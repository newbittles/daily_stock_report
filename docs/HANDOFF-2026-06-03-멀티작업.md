## 4. 공통 — cross_signal SSOT (충돌 해소 결과)

세 작업에 혼재했던 cross_signal을 **단일 기준으로 통일**(이 세션 머지 시):
- **SSOT = `patterns.core.ma_cross_signal(closes)` + `CROSS_PULLBACK`/`CROSS_CORRECTION` 상수**
- 기준: 5<10 데드크로스 + 20일이격 ≥15%→PULLBACK(🟢매수기회) / ≤7%→CORRECTION(⚠️경고)
- 사용처: 한국 `strategy_section`(리포트 표시) · 미국 `us_pipeline`(배지) · 자동매매 `decide_exit`(청산)
