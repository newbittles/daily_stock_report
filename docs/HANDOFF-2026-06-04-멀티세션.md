  main/src 사용(수정 미반영). 이번 세션은 main repo 직접 작업이라 무관.
- **테스트**: `.venv\Scripts\python.exe -m pytest tests/ -q` → 150 통과 기준선.
- **콘솔 인코딩**: Windows cp949 — 한글 검증결과는 utf-8 파일로 써서 Read.
- **GitHub Pages**: push마다 재빌드(~30-60s), 시간당 소프트제한 → 불필요한 재발행 자제.
- **KIS 엔드포인트/TR**: 공식 문서 검증 없이 하드코딩 금지(§6).
- **커밋 메시지**: PowerShell here-string(`@'..'@`)이 Bash 툴에서 '@' 누출/실패 → `git commit -F 파일`
  또는 Bash heredoc(`<<'EOF'`) 사용. (초기 커밋 cb98a67/a2bacaf 메시지에 '@' 접두 누출됨, 무해.)
